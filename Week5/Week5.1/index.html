New Hope Programming Bootcamp

This website serves as a collection of projects that have come out of New Hope Programming Bootcamp in Summer 2020.

About

The New Hope Programming Bootcamp is a program led by New Hope Youth Leadership that teaches the basics of building a website.

Vision Statement

To see every member of New Hope Programming Bootcamp cultivate and use their God-given gifts for the glory of God.

Mission Statement

Inspire each learner to make an impact, support each learner in discovering their God-given gifts, and create a supportive community of learners.

Resources

All curriculum of New Hope Programming Bootcamp is available online on GitHub, Google Slides, and YouTube.
